Quiz 03
=======  

Question 01  
-----------  
a, b,and c are three integer type variables. If b=21 and c = 7, what are the values of a, b, and c after the execution of the statement a = b + c * c % b++;  

### Answer  
a = 28, b = 22, c = 7  

Question 02
-----------  
a, b,and c are three integer type variables. If b=21 and c = 7, what are the values of a, b, and c after the execution of the statement a = (b + c) * c % b++;  

### Answer  
a = 6, b = 22, c = 7
